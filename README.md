# Conway

This is [Conway's Game of
Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) in React.js and
TypeScript.

https://conway-react.herokuapp.com/

### Setup

```
$ git clone https://github.com/jwworth/conway.git
$ cd conway
$ yarn
$ yarn start
```

### Testing

```
$ yarn test
```
